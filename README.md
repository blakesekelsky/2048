# 2048
2048 built with SDL2, uses SDL2-dev and SDL2-ttf

## Dependencies:
 * ubuntu: ```sudo apt install clang libsdl2-dev libsdl2-ttf-dev```

## Making executable
### Development:
* build binary ```make dev_binary```
* build and run ```make run_dev```
### CMake:
* build cmake ```make cmake```
* build and run ```make run```
